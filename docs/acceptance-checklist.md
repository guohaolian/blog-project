# 验收清单（QA Checklist）

> 与 `docs/prd.md` 对应，用于自测/联调/上线前检查。

## A. 后台管理端（Admin Web）

### A1 登录与鉴权
- [ ] 输入正确账号密码可登录
- [ ] 退出登录后 token 清理，访问受保护页面会跳转登录
- [ ] token 失效：任意接口返回 401，前端能自动回登录
- [ ] 403：提示无权限，但不清 token

### A2 文章管理
- [ ] 新建草稿成功
- [ ] 草稿发布成功，前台可见
- [ ] 编辑文章成功，前台同步变化
- [ ] 删除文章后前台不可见
- [ ] 列表分页、搜索、状态筛选可用

### A3 分类/标签
- [ ] 新增/编辑/删除分类标签可用
- [ ] 分类/标签与文章关联正确

### A4 上传/资源
- [ ] 上传封面成功，返回可访问 URL
- [ ] 上传正文图片成功
- [ ] 资源列表可看到已上传资源（如落库）
- [ ] 删除资源后不可再访问（或至少资源列表不显示）

### A5 评论（含可选需求）
- [ ] 评论列表可看到待审核/已通过/已拒绝
- [ ] 审核通过后前台可见
- [ ] 审核拒绝后前台不可见
- [ ] 删除评论后前台不可见

### A6 站点设置（含可选需求）
- [ ] 修改站点名/公告/关于/SEO/页脚后前台展示更新

---

## B. 前台 Web

### B1 浏览
- [ ] 首页列表分页正常
- [ ] 文章详情可打开
- [ ] 分类/标签筛选正常
- [ ] 搜索正常
- [ ] 归档页可用
- [ ] 热门文章可用（按阅读量）

### B2 SEO/路由
- [ ] 刷新任意路由不 404（Nginx SPA fallback 正常）
- [ ] 页面 title/description 基础设置有效

### B3 评论
- [ ] 可提交评论，默认提示“待审核”
- [ ] 审核通过后展示

---

## C. API

- [ ] 统一响应体符合 `docs/api-conventions.md`
- [ ] 参数校验失败行为符合约定（400 或 code=40001）
- [ ] 401/403/500 行为符合约定

---

## D. 部署

- [ ] 前台域名可访问
- [ ] 后台域名可访问
- [ ] `/api` 反代可用
- [ ] 上传图片在公网可访问（/uploads 映射）
- [ ] systemd 后端服务：开机自启、崩溃重启
- [ ] 数据库备份定时任务存在并可产出文件
